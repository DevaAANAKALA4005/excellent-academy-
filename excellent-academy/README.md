# Excellent Academy – Best School in Mandleshwar

A modern, premium, fully responsive school website built with pure HTML, CSS and JavaScript. SEO-optimized for the local keyword **"Best School in Mandleshwar"** and ready to deploy on GitHub Pages.

## Pages

- `index.html` – Home (Hero, About, Why Choose, Facilities, Academics, Kindergarten, Testimonials, FAQ, Contact)
- `about.html` – About Us
- `academics.html` – Academic programs & Science Stream
- `admissions.html` – Admissions inquiry & process
- `facilities.html` – Infrastructure & facilities
- `gallery.html` – School photo gallery
- `contact.html` – Contact info + form + Google Maps
- `best-school-in-mandleshwar.html` – Dedicated SEO landing page (1200+ words)

## Files

- `css/styles.css` – Full responsive stylesheet (blue / white / gold theme)
- `js/script.js` – Mobile menu, animated counters, scroll reveal, FAQ, form handling
- `sitemap.xml` – XML sitemap for search engines
- `robots.txt` – Search engine directives

## Deploy to GitHub Pages

1. Create a new GitHub repository (public).
2. Upload the entire `excellent-academy/` folder contents to the repo root.
3. Go to **Settings → Pages**.
4. Select branch `main` and folder `/` (root). Save.
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
6. Update the canonical/Open Graph URLs in each HTML file from `https://excellentacademy.example.com/` to your live URL.
